# ✨ BOB — Sphère WebGL ✨

Projet test pour la démonstration WebGL / Three.js.

## 📄 Index.html

Page unique qui affiche :
- **Sphère extérieure** : Wireframe cyan à rotation continue
- **Sphère intérieure** : Sphère solide bleue semi-transparente (rotation opposée)
- **Particules flottantes** : Nuage de points verts animés en sinusoïdale
- **Background** : Dégradé bleu nuit élégant

## 🎨 Couleurs

| Élément | Couleur |
|---------|---------|
| Sphère ext | Cyan (#00ffff) |
| Sphère int | Bleu (#0088ff) |
| Particules | Vert (#00ff88) |

## 🚀 Démarrage

Ouvrir simplement `index.html` dans un navigateur.

Le projet utilise Three.js via CDN (pas d'install npm requise).

## 📦 Dépendances

- **Three.js** r128 via [cdnjs](https://cdnjs.cloudflare.com/ajax/libs/three.js/r128/three.min.js)

## 🔧 Technologies

- WebGL 2.0
- Three.js
- HTML5 Canvas
- ES6+
